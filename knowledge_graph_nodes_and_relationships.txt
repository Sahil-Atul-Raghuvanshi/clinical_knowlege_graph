NOTES:
======
- Patient nodes are linked to HospitalAdmission and Discharge nodes via subject_id property (no explicit relationship edge)
- Event flow relationships (LED_TO_*) are created dynamically between event nodes (EmergencyDepartment, UnitAdmission, ICUStay, Discharge) based on chronological order
- ICUStay nodes are created by converting UnitAdmission nodes, so they inherit the same relationships
- All batch nodes (ProceduresBatch, PrescriptionsBatch, LabEvents, ChartEventBatch) serve as containers for their respective child nodes
